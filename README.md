# graphql-js-starter

## Start the project

### Clone the project

```
cd <your_workspace>/
git clone https://github.com/jpmai/sarl-bonnevial.git
cd sarl-bonnevial/
```

### Run the project

```
npm install           // or yarn
npm run start         // or yarn start
```
